Production-ready REST API template with JWT authentication, role-based access control, rate limiting, and async SQLAlchemy. Built as a starting point for new projects -- clone it, define your models, and start building endpoints.

All protected endpoints require a Bearer token. Get one via /auth/login: 

curl -X POST /auth/login \
  -H "Content-Type: application/json" \
  -d '{"email": "demo@example.com", "password": "demo1234"}'

# Use the access_token in subsequent requests:
curl /items/ -H "Authorization: Bearer <access_token>"
